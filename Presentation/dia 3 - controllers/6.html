3.3 | Http Verbs

GET

O método GET é usado para recuperar informações de um servidor através de uma URL.

Requests usando GET devem somente recuperar dados e não devem ter nenhum outro efeito nos dados.

POST

Uma request POST é usada para enviar dados para o servidor

Geralmente usando forms html

PUT

Substitui os dados do destino com o conteúdo enviado

DELETE

Remove os dados do destino de uma URL
