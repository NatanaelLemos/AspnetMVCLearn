1.1 | Front-End VS Back-End

Vamos falar do Front-End

HTML: Hypertext Markup Language (Linguagem de Marcação de Hipertexto), linguagem base/principal para progração web.

CSS: Cascading Style Sheets (folhas de estilo em cascata), utilizada para determinar o visual da página.

JavaScript: Linguagem de programação baseada em scripts (utilizada para validações, animações, ajax).
